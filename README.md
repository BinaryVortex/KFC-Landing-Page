# KFC Landing Page

A clean, responsive landing page inspired by KFC — built using plain HTML and CSS for practice and demonstration purposes.

![KFC Landing Page Preview](Screenshot%202024-11-13%20060704.png)

## Demo
Open `index.html` in your browser to view the landing page locally.

## Features
- Pixel-perfect HTML & CSS layout
- Responsive header and hero section
- Clean typography and spacing
- Static demo suitable for learning and customization

## Built With
- HTML
- CSS

## Getting Started
To run this project locally:

1. Clone the repository:

```bash
git clone https://github.com/BinaryVortex/KFC-Landing-Page.git
cd KFC-Landing-Page
```

2. Open the `index.html` file in your browser (double-click or use a local server):

```bash
# using Python 3's simple HTTP server
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```
